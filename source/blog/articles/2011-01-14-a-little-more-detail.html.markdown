Still curious about KidsRuby? We wrote a post explaining in a little more detail what it's about called ["A little more detail"](http://www.kidsruby.com#2011-01-14-2) on the KidsRuby site.

KidsRuby provides a way that kids can use the awesome Ruby programming language with a simple interface that lets them explore it's full capabilities on any computer, regardless of operating system.
